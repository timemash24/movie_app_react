## 사용 스택
React.js, Javascript, HTML, CSS, Sass(SCSS)

## 기능 설명
- 영화 목록 표시
  - 영화 API를 이용하여 평점 7점 이상 영화 다운 많은 순으로 영화 목록 표시
  - 선택한 장르에 해당하는 영화로 분류하여 표시
  - 하트 아이콘 클릭으로 마음에 드는 영화 localstorage에 저장/해제 
  
- 영화 상세 정보
  - 영화 사진 클릭 시 상세정보 페이지 이동
  - 선택한 영화와 연관된 영화 추천
 
- 개인 페이지
  - localstorage에서 마음에 드는 영화 불러오기
  - 장르에 따라 분류 가능
