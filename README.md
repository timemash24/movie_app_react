## 사용 스택
React.js, Javascript, HTML, CSS

## 기능 설명
- 영화 목록 표시
  - 영화 API를 이용하여 평점 9점 이상 영화 불러오기
  
- 장르별 분류
  - 장르에 해당하는 영화 목록 표시
  
- 영화 상세 정보
  - 영화 사진 클릭 시 상세정보 페이지 이동
